# LabSafeEdu实验室安全智能体原型系统

## 一、项目简介

LabSafeEdu是一款面向高校化学与材料类实验室的安全教育与风险闭环管理智能体原型系统。系统围绕学生准入、实验前风险评估、隐患识别整改、准入考试评价和安全管理数据分析等场景，提供安全学习、风险研判、整改建议和数据看板服务。

本版本为本地运行原型，不调用任何付费API，适合参赛演示、录屏和功能验证。

## 二、功能模块

1. 首页：展示系统定位、核心数据和闭环逻辑。
2. 安全准入学习：根据用户身份和实验室类型生成入室前学习清单。
3. 实验前风险评估：根据实验风险因素生成风险等级、防护要求、审批备案提醒和检查清单。
4. 隐患识别与整改：根据隐患描述判断隐患类别、风险等级、整改措施和复查要点。
5. 准入考试与错题反馈：展示题库、答案解析和薄弱点复习建议。
6. 安全数据分析看板：展示832项隐患数据、风险等级分布和管理建议。
7. 参赛材料导出：提供简介、关键词、创新点和演示摘要。

## 三、安装与运行

### 1. 安装依赖

在项目文件夹中打开终端，运行：

```bash
pip install -r requirements.txt
```

### 2. 启动系统

```bash
streamlit run app.py
```

启动后浏览器会自动打开本地网页。
