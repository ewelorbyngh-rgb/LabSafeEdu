# LabSafeEdu Final Smart FIX4

正式参赛展示用智能体系统。该版本对所有模块做了智能化自查和增强：
- AI问答优先识别应急处置、隐患整改、实验前风险评估和知识问答。
- 隐患识别具备兜底判断，不依赖完全预置句子。
- 实验前风险评估支持文本和勾选因素综合判断。
- 准入学习根据身份和实验室类型生成方案。
- 准入考试支持错题知识点复习联动。
- 移除参赛材料导出等开发辅助模块。

Run:
python -m pip install -r requirements.txt
python -m streamlit run app.py


## MenuClean 修正记录
- 将“AI安全问答助手”更名为“智能问答与应急助手”。
- 将“安全知识库”更名为“制度案例知识库”。
- 目的：区分“交互式智能助手”和“知识资料支撑库”，避免评委认为模块重复。


## FIX5 知识库维护增强
- 制度案例知识库基于本地 data/ 文件夹中的 CSV 数据构建。
- 增加“管理者更新”入口，可下载当前数据备份、下载空白模板、上传 CSV/Excel 批量更新。
- 增加“快速新增一条制度知识”表单，便于管理者及时补充学校或学院新制度。
- 本地运行时可直接保存到 data/ 文件；线上 Streamlit Cloud 演示环境的写入可能在重启后丢失，长期线上使用建议同步到 GitHub 或接入数据库。


## FIX6 任意格式资料智能导入
- 在“制度案例知识库—管理者更新”中增加“资料智能导入更新”。
- 支持上传 Word、PPT、Excel、CSV、PDF、TXT、Markdown、图片等资料。
- 系统自动解析文字/表格，拆分条目，并转换为制度知识库、隐患整改库、风险规则库、问答库、案例微课堂所需结构。
- Excel/CSV若表头匹配，可直接入库；非结构化文档会自动分类生成条目。
- 图片OCR依赖运行环境是否安装 OCR 引擎；若云端未启用OCR，可将图片文字粘贴到补充文字框完成同样转换。
